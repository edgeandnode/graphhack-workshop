# Graph Hack Workshop

## Requirements

- **[PNPM] [7](https://github.com/pnpm/pnpm/releases/tag/v7.0.0)**
- **Node >=14**

[pnpm]: https://pnpm.io/

## Contributing

1. Install dependencies with **`pnpm install`**
2. Run **`pnpm codegen`** to generate types from GraphQL queries
3. Run **`pnpm build`** to build the packages.
